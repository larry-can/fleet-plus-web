Funcionalidades principales

Vehículos

Gestiona toda tu flota de forma clara y ordenada. Genera informes completos por vehículo.

Componentes

Crea el listado que desees con los tipos de componentes que mejor se adapte a tus necesidades.

Productos

Crea, modifica y elimina cualquier producto, organizados por tipo de componente.

Mantenimientos

Registra cualquier intervención en la flota (tipo de componente utilizado, producto, kilómetro en en el que se hizo, etc.).

Obligaciones

Registra tareas inherentes a los vehículos (ITV, vigencia de seguros, permiso de circulación,...)

Proveedores

Archiva información completa de talleres, aseguradoras y empresas de toda clase de servicios.

Facturas

Control económico por medio de facturas; relacionando vehículo, proveedor, número de factura, fecha e importe.

Gastos

Todo el coste asociado a la flota, organizada por categoría y concepto, pudiendo estar o no relacionada con una factura.

Instalación y Ejecución

De momento, hasta que no esté preparado el ejecutable de fleet+, sólo existe una forma de usar el software y es ejecutándolo desde el código fuente.

Desde el Código Fuente

Método ideal para contribuir o usar la versión más reciente.

1. Requisitos previos

  • Python 3.10+
  • Git

2. Clonar el repositorio

git clone https://github.com/larry-can/fleet_plus.git
cd fleet_plus

3. Crear entorno virtual e instalar dependencias

python -m venv venv
source venv/bin/activate   # Linux/macOS
venv\Scripts\activate      # Windows

pip install -r requirements.txt

4. Ejecutar fleet+

python app.py

Guía de Funcionalidades

fleet+ se organiza en varios módulos accesibles desde la interfaz principal:

  • Vehículos: gestión completa de cada vehículo, historial e informes en PDF.
  • Componentes: define cualquier tipo de componente usado por la flota.
  • Productos: catálogo organizado, marcas, modelos y tipo.
  • Mantenimientos: registra intervenciones, kilometraje y componente utilizado.
  • Obligaciones: control de ITV, seguros y documentación.
  • Proveedores: talleres, aseguradoras, empresas de servicios.
  • Facturas: importes, fechas y relación con vehículos.
  • Gastos: control económico general, categorías y totales.
